# Course Management System

## 📌 Project Overview

The Course Management System is a web-based application designed to manage courses, students, and enrollments efficiently. It helps administrators organize course information and allows users to access course-related data easily.

This project demonstrates the use of backend logic, database management, and web technologies to build a simple course management platform.

## 🚀 Features

* Add new courses
* View available courses
* Student registration
* Course enrollment management
* Update course information
* Delete courses
* Database integration

## 🛠 Technologies Used

* Java
* JSP
* Servlets
* JDBC
* MySQL
* HTML
* CSS
* Apache Tomcat Server

## 🏗 Project Structure

* **DTO Classes** – Data transfer objects (User, Course, Enrollment)
* **DAO Classes** – Database operations
* **JSP Pages** – User interface
* **Servlets** – Request handling and business logic
* **Database** – MySQL for storing course and user data

## ⚙ How to Run the Project

1. Install Java and Apache Tomcat.
2. Import the project into an IDE (Eclipse / VS Code).
3. Configure the MySQL database.
4. Update database connection in the project.
5. Deploy the project on Tomcat Server.
6. Open the application in a web browser.

## 🎯 Purpose of the Project

This project was developed to understand the concepts of:

* Web application development
* MVC architecture
* Database connectivity using JDBC
* Backend and frontend integration.
